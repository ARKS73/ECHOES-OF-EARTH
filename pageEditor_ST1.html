Splash Page

Technical Description

HTML Techniques

  • Used for automatic redirection after 5 seconds
  • Created a centered content container using (div class="main") with absolute positioning
  • Implemented semantic headings (h1, h3, h4) for clear hierarchy
  • Added an animated loading indicator with (div class="loader")

CSS Techniques

  • Full-screen background image with background-size: fill
  • Centered content using transform: translate(-50%, -50%)
  • Created visual contrast with semi-transparent overlay (background-color: rgba(0,0,0,0.4))
  • Added CSS animation (@keyframes spin) for the loading spinner
  • Implemented hover effects for interactive elements (:hover)

Key Properties

position: absolute, transform, @keyframes, background-size, border-radius

Accessibility

  • High contrast text (white on dark semi-transparent background)
  • Clear heading structure for screen readers
  • Added manual navigation option (button) alongside automatic redirect
  • Semantic HTML structure for better screen reader interpretation

Challenges and Lessons Learned

Form accessibility

  • Solution: Added proper labels and ARIA attributes.
  • Lesson: Accessibility should be considered from start.

Initially struggled to implement a CSS-only loading animation

  • Solution:After researching CSS animations on W3Schools
  • Lesson: Learned that CSS animations (via @keyframes) are more performant than JavaScript alternatives for simple animations.

Compliance

  • All images are either original or properly licensed
  • Followed W3C standards for HTML and CSS
  • Implemented basic accessibility features
  • Avoided automatic content that could disorient users
